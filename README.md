# 🩺 看護師向け計算ツールアプリ（nurse-calc-app）

看護師業務でよく使われる薬剤計算・点滴速度・体液バランスなどの計算を、スマートに行えるWebアプリです。
スマートフォンでも使いやすいUIで、忙しい現場でもすぐ使えることを目指しています。

**デモ：[こちらからお試しできます](https://nurse-calc-app.vercel.app)**

- [📘 Zenn（開発背景・UI/UX解説）](https://zenn.dev/moch1/articles/d9d8454697d1fe)

## 🚀 主な特徴

- シンプルで見やすいインターフェース（ナイトモード対応）
- 9種類の計算機能を搭載
- 計算履歴の自動保存（各カテゴリ最大10件）
- 履歴から入力値を再利用して再計算可能
- ログイン不要・インストール不要ですぐ使える
- 初回利用時のオンボーディングガイド表示

## 🧮 実装済みの計算機能

| 機能                | 説明                                    |
| ------------------- | --------------------------------------- |
| 💊 投薬計算         | 必要量・投与速度を体重・濃度から算出    |
| 💧 点滴速度計算     | 投与量・時間から点滴速度（滴/分）を計算 |
| ⚖️ 体液バランス計算 | IN/OUTを集計しバランスを算出            |
| 🥗 栄養計算         | 体重から必要カロリー・たんぱく量を計算  |
| 📐 体表面積（BSA）  | Mosteller式でBSAを計算                  |
| 🧪 電解質補正       | Na・K・Mgの補正量を計算                 |
| 🫁 酸素投与量計算   | デバイス・流量から推定FiO₂を算出        |
| 🩸 輸血計算         | 目標Hb値から輸血量を計算                |
| 📊 BMI計算          | 身長・体重からBMI・肥満度を算出         |

## 📁 ディレクトリ構成（主要部分）

```
src/
├── app/
│   ├── _components/      # 共通UIコンポーネント
│   ├── _context/         # テーマ（ナイトモード）コンテキスト
│   ├── bmi/              # BMI計算
│   ├── bsa/              # 体表面積計算
│   ├── drip/             # 点滴速度計算
│   ├── electrolyte/      # 電解質補正計算
│   ├── fluid/            # 体液バランス計算
│   ├── history/          # 履歴一覧
│   ├── medication/       # 投薬計算
│   ├── nutrition/        # 栄養計算
│   ├── oxygen/           # 酸素投与量計算
│   └── transfusion/      # 輸血計算
├── config/               # デバイス設定・コンポーネントマッピング
├── hooks/
│   └── useCalculator.ts  # 計算機共通フック（RHF + Zod）
├── lib/                  # 履歴・並び順・再利用ユーティリティ
└── types/                # 共通型定義
```
